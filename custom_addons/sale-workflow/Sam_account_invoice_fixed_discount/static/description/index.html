Sale Workflow - Account Invoice Fixed Discount

This module adds a fixed discount to the total amount of the invoice.

To use this module, you need to:

  1. Install the module.
  2. Go to Accounting > Configuration > Sales > Accounting Profiles.
  3. Select the accounting profile that you want to use with this module.
  4. Go to the "Invoicing" tab and check the "Apply Fixed Discount" checkbox.
  5. Enter the fixed discount amount in the "Fixed Discount" field.
  6. Save the accounting profile.

When you create an invoice, the fixed discount will be applied to the total amount of the invoice.
